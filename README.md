# CS4120FinalProject
Final Project for Natural Language Processing by William Sun, Andrew Lee, Suneet Pathangay, and Alexander Tu

# Direction 

## Step 1: 

Download all the required packages in requirements.txt 

## Step 2: 
Run `data_preparation.ipynb` to generated cleaned csv data file 

## Step 3: 
Run `sentiment_analysis` for VADER, FinBERT model predictions (it will generate csv files required for next steps)

## Step 4: 
Run `roberta_analysis` for RoBERTa model predictions (it will generate csv files required for next steps)

## Step 5: 
Run `logistic_regression_analysis.ipynb` for Logical Regression model predictions (it will generate csv files required for next steps)

## Step 6: 
Run `statistical_analysis.ipynb` for statistical analysis part
